Zum Starten des Programms GrooveLoad.vbs ausführen.

Das Programm wurde in AutoIt geschrieben. Es ist keine EXE, lediglich der Sourcecode vorhanden,
dieser wird durch den beiliegenden Interpreter (Data\AutoIt3.exe) auch auf Systemen ohne AutoIt
Installation ausgeführt. Der Interpreter stammt aus AutoIt 3.3.12.0.
Mehr über AutoIt findest du unter autoitscript.com. Die Lizenz (des Interpreters) kann unter
http://www.autoitscript.com/autoit3/docs/license.htm eingesehen werden.

Die Credits sind im Programm unter dem Punkt "Über" zu finden. Das Änderungsprotokoll kann unter
hhttp://hegi.pfweb.eu/grooveload/download/Changelog.txt eingesehen werden.

-----

You have to unpack the program from the ZIP archive!
To start the program run GrooveLoad.vbs.

The program is written in AutoIt. There is only the source code available (no EXE) which will be
executed by the included interpreter (Data\AutoIt3.exe). The interpreter comes from AutoIt 3.3.12.0
More about AutoIt can be found under autoitscript.com. The license (of the interpreter) can be
viewed at http://www.autoitscript.com/autoit3/docs/license.htm.

The credits can be found in the program under "About". The changelog can be viewed under
http://hegi.pfweb.eu/grooveload/download/Changelog.txt.
